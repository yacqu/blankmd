# Welcome to blankmd

A distraction-free Markdown editor that lives in your browser.

## Why blankmd?

Writing should be **simple**. No accounts, no cloud sync, no distractions — just you and your words. Everything saves locally, works offline, and stays *yours*.

## Features

* **Offline-first** — works without internet
* **Local storage** — your data never leaves your device
* **Custom themes** — make it yours
* **Keyboard shortcuts** — write faster

## Quick Example

Here's some code to get you started:

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("world"));
```
